# Superstore Sales Dashboard (Power BI)

## Project Overview
This Power BI project analyzes Superstore sales data to uncover insights into sales performance, profit trends, and customer behavior.

## Tools Used
- Power BI
- Excel (Data Source)

## Key Insights
- Sales and profit trends by region
- Top-performing categories and sub-categories
- Impact of discounts on profit
- Monthly and yearly sales analysis

## Dashboard Features
- Interactive slicers (Region, Category, Year)
- KPI cards for Sales, Profit, Orders
- Visuals: Bar charts, Line charts, Maps
